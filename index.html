    • LibPKMN

    Universal Pokémon save editing and analysis library

    Project maintained by Nicholas Corgan Hosted on GitHub Pages — Theme by mattgraham

    Description

    LibPKMN is a C++ library that allows users to read and edit game saves of any Pokémon game from the main series (22 games released 1996-2013). With LibPKMN, users can generate Pokémon teams on their PC and import them into their saves, allowing their creations to be used in-game. Even without a save present, users can use the library for team study and strategy formation.

    This library is designed to be as portable as possible. Using the CMake build system for platform-independence. LibPKMN can be compiled with any modern version of GCC, Clang, or Microsoft Visual Studio C++. Portability is further extended through the use of SWIG, which generates Python, C#, and Java API's to allow further development.

    Submodules

    LibPKMN would not exist without the inclusion of other open-source libraries, included as Git submodules.

    • SQLiteCpp - SQLite3 C++ wrapper
    • Pokehack - Generation III save editing library
    • PokeLib-NC - Generation IV save editing library, a fork of Robert Maupin's PokeLib
    • PKMDS - Generation V-VI save editing library
